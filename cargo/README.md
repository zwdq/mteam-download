# M-Team 种子查询工具 (Rust 版本) 🦀

M-Team 种子查询工具的高性能 Rust 实现，提供与 Python 版本相同的功能，但具有更好的性能和更小的分发体积。

## ✨ 核心特性

- 🚀 **高性能**：编译为原生代码，启动速度快
- 🔐 **Token 认证**：使用 M-Team 官方 API Key 进行安全认证
- 🔍 **强大搜索**：支持关键词搜索、分类筛选、多维度排序
- 📥 **种子下载**：支持直接下载种子文件到本地
- 📂 **分类浏览**：按分类浏览热门资源
- 🛡️ **代理支持**：内置代理配置，完美绕过 Cloudflare 拦截
- 📊 **智能排序**：自动识别中文字幕资源并优先展示
- 💎 **类型安全**：Rust 保证内存安全和类型安全
- 📦 **单文件分发**：编译后无需依赖，直接运行

## 📋 系统要求

- Rust 1.70+ (编译)
- 网络连接 (建议配置代理)

## 🚀 快速开始

### 1. 编译项目

```bash
cd cargo
cargo build --release
```

编译后的二进制文件位于 `target/release/mteam-query.exe` (Windows) 或 `target/release/mteam-query` (Linux/macOS)

### 2. 配置 API Key

复制上级目录的配置文件：
```bash
cp ../config.yaml .
```

或创建新的 `config.yaml`：

```yaml
api_key: "your_mteam_api_key_here"

proxies:
  http: "http://127.0.0.1:7890"
  https: "http://127.0.0.1:7890"

timeout: 15
```

### 3. 获取 API Key

登录 M-Team 站点 → 进入设置 → API 管理 → 生成 API Key

## 📖 使用方法

### 🎮 交互式模式 (推荐)

直接运行启动交互式界面：

```bash
./target/release/mteam-query
# 或
cargo run --release
```

**交互式功能：**

#### 📋 主菜单
```
============================================================
📋 主菜单
============================================================
  1. 🔍 搜索种子
  2. 📂 按分类浏览
  3. ⚙️  设置
  0. 🚪 退出
============================================================
```

#### 🔍 快速搜索
- 直接输入关键词搜索种子
- 可选按分类筛选
- 选择排序方式 (做种数/文件大小/发布时间)
- 设置返回结果数量
- 支持下载种子、获取链接、查看详情

#### 📂 分类浏览
- 按分类查看热门资源
- 支持电影、剧集、动漫、综艺、体育等分类
- 每个分类显示做种数最多的资源

#### 📥 下载功能
搜索结果支持以下操作：
- **下载种子文件**：直接下载 .torrent 文件到本地
- **获取下载链接**：复制链接用于下载工具
- **查看详情**：显示种子的完整信息

### 💻 命令行模式

#### 基本搜索

```bash
# 搜索种子
./mteam-query search "沙丘2"

# 限制返回数量
./mteam-query search "沙丘2" -l 10

# 按分类搜索 (401=SD 电影)
./mteam-query search "电影" -c 401

# 按大小排序
./mteam-query search "沙丘" -s size
```

#### 查看详情

```bash
# 查看种子详细信息
./mteam-query detail 123456
```

#### 获取下载链接

```bash
# 获取种子下载链接
./mteam-query url 123456
```

#### 下载种子文件

```bash
# 下载种子文件到当前目录
./mteam-query download 123456

# 下载到指定路径
./mteam-query download 123456 -p /path/to/save
```

## 🎨 输出说明

### 中文字幕标记
搜索结果中带 💖 标记的表示包含中文字幕或国配资源，会优先展示。

### 搜索结果字段
- **序号**：用于选择和下载
- **标题**：中文标题（优先）或英文文件名
- **大小**：格式化的文件大小
- **做种**：当前做种人数
- **下载**：当前下载数
- **分类**：M-Team 分类名称
- **种子页面链接**：可直接访问的详情页链接

## 🔧 技术栈

- **HTTP 客户端**: reqwest (异步)
- **序列化**: serde, serde_json, serde_yaml
- **命令行**: clap (derive)
- **交互界面**: dialoguer, console
- **运行时**: tokio
