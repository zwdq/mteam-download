# M-Team 种子查询工具 🐱

一个使用 Python 编写的专业 M-Team 站点种子查询工具,支持通过 API Token 进行高效的种子搜索、详情查询和下载。

## ✨ 核心特性

- 🎮 **交互式终端**: 友好的交互式菜单界面,支持搜索、浏览和下载
- 🔐 **Token 认证**: 使用 M-Team 官方 API Key 进行安全认证
- 🔍 **强大搜索**: 支持关键词搜索、分类筛选、多维度排序
- 📥 **种子下载**: 支持直接下载种子文件到本地
- 📂 **分类浏览**: 按分类浏览热门资源
- 🛡️ **代理支持**: 内置代理配置,完美绕过 Cloudflare 拦截
- 📊 **智能排序**: 自动识别中文字幕资源并优先展示
- 💎 **SOLID 设计**: 遵循 S.O.L.I.D 原则的代码架构

## 📋 系统要求

- Python 3.7+
- 网络连接 (建议配置代理)

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

1. 复制配置文件模板:
```bash
cp config.example.yaml config.yaml
```

2. 编辑 `config.yaml`,填入你的 M-Team API Key:
```yaml
api_key: "your_mteam_api_key_here"

proxies:
  http: "http://127.0.0.1:7890"
  https: "http://127.0.0.1:7890"
```

### 3. 获取 API Key

登录 M-Team 站点 → 进入设置 → API 管理 → 生成 API Key

## 📖 使用方法

### 🎮 交互式模式 (推荐)

直接运行启动交互式界面:

```bash
python mteam_query.py
# 或
python mteam_query.py -i
```

**交互式功能:**

#### 📋 主菜单
```
============================================================
📋 主菜单
============================================================
  1. 🔍 搜索种子
  2. 📂 按分类浏览
  3. ⚙️  设置
  0. 🚪 退出
============================================================
```

#### 🔍 搜索模式
- 输入关键词搜索种子
- 可选按分类筛选
- 选择排序方式 (做种数/文件大小/发布时间)
- 设置返回结果数量
- 支持下载种子、获取链接、查看详情

#### 📂 分类浏览
- 按分类查看热门资源
- 支持电影、剧集、动漫、综艺、体育等分类
- 每个分类显示做种数最多的资源

#### 📥 下载功能
搜索结果支持以下操作:
- **下载种子文件**: 直接下载 .torrent 文件到本地
- **获取下载链接**: 复制链接用于下载工具
- **查看详情**: 显示种子的完整信息

### 💻 命令行模式

#### 基本搜索

```bash
# 搜索种子
python mteam_query.py search "沙丘2"

# 限制返回数量
python mteam_query.py search "沙丘2" -l 10

# 按分类搜索 (401=SD 电影)
python mteam_query.py search "电影" -c 401

# 按大小排序
python mteam_query.py search "沙丘" -s size
```

#### 查看详情

```bash
# 查看种子详细信息
python mteam_query.py detail 123456
```

#### 获取下载链接

```bash
# 获取种子下载链接
python mteam_query.py url 123456
```

## 🌟 使用示例

### 交互式模式示例流程

1. **启动程序**
```bash
python mteam_query.py -i
```

2. **选择搜索功能** (输入 `1`)

3. **输入关键词** (如: `沙丘2`)

4. **选择排序方式** (推荐选择做种数)

5. **查看搜索结果** (含中文字幕标记 💖)

6. **下载种子** (输入序号 → 选择下载功能)

7. **输入保存路径** (或使用默认当前目录)

8. **完成下载** ✅

## 🎨 输出说明

### 中文字幕标记
搜索结果中带 💖 标记的表示包含中文字幕或国配资源,会优先展示。

### 搜索结果字段
- **序号**: 用于选择和下载
- **标题**: 种子名称
- **大小**: 格式化的文件大小
- **做种**: 当前做种人数
- **下载**: 当前下载数
- **分类**: M-Team 分类 ID
- **种子页面链接**: 可直接访问的详情页链接
